# 🎓 CikguBot  
### AI Learning Assistant with Quiz System

## 📌 Overview
CikguBot is an AI-powered learning assistant designed to support primary school students through interactive tutoring and gamified quizzes.

The system combines **conversational learning** with **AI-generated quizzes**, making studying more engaging and effective.

---

## 🚀 Features

- 💬 AI conversational tutor for science topics  
- 🧠 AI-generated quiz questions  
- 🎮 Gamification system (points, stickers, achievements)  
- 📊 Interactive learning experience for students  

---

## 🧠 Problem Statement
Many students face challenges such as:
- Lack of engagement in traditional learning  
- Difficulty understanding complex topics  
- Limited personalized learning support  

---

## 💡 Solution
CikguBot provides:
- An interactive AI tutor for guided learning  
- Dynamic quiz generation to reinforce understanding  
- Gamified elements to increase motivation and retention
